Title

The title of our project is Point Cloud to Mesh

Summary

Our project aims to convert point clouds into 3D meshes. Specifically, given an input file of format .plz our program will output a 3D triangle mesh in the format .dae. For our test files, we will be using the Stanford 3D scanning repository.

Background

The problem we are trying to solve is that, given a set of points (a point cloud), how do we generate a 3D mesh from that pointcloud which connects all of the points. Mathematically, we are trying to create a function, which, given a set of points, $P$, such that $$P \subset {\rm I\!R}^3$$ Describe a function $f$, which will convert this point cloud to a mesh. $$f : {P | P \subset {\rm I\!R}^3} \rightarrow {M | M \subset }$$
