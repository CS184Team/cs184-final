Title, Summary, and Team Members

The title of our project is Point Cloud to Mesh. Our project aims to convert point clouds into 3D meshes. Specifically, given an input file of format .plz our program will output a 3D triangle mesh in the format .dae.

Background

The problem we are trying to solve is that, given a set of points (a point cloud), how do we generate a 3D mesh from that pointcloud which connects all of the points.
